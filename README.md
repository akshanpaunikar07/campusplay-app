# campusplay-app
This project is a simple UI design concept for a Campus Sports Booking platform.
It includes clean and minimal interfaces for:

User Login
Sports Facility Selection
Navigation & Step-by-step booking flow
The goal of this project is to practice UI layout, visual hierarchy, spacing, and building user-friendly screens.

Tools Used:

Canva basic UI tools
GitHub for version control
